\# TeamPulse - Weekly Report System



TeamPulse is a full-stack web application designed for team members to submit weekly work reports and for managers to track team progress through a consolidated dashboard.



\## Prerequisites



Before running the application, ensure you have the following installed:

\- Node.js (v18+)

\- Java (JDK 17+)

\- MySQL (XAMPP or local server)



\## Setup Instructions



\### 1. Database Setup

1\. Open XAMPP and start MySQL.

2\. The application is configured to automatically create the database `weekly\_report\_db` on startup.

3\. Ensure your MySQL credentials match those in `backend/src/main/resources/application.properties`.
